# 搭建开发环境
您必须要搭建一个开发环境才能构建 STM32 项目。这个环境包含了必要的编译器、库，以及向微处理器编译和烧录固件时必须用到的工具。STM32Tesseract GUI 提供了一个便捷的方法来搭建该环境。

## 准备条件
在构建开发环境前，您需要确保您已满足以下条件 :
- 一个稳定的网络连接环境
- 足够的磁盘空间（大约 1 GB）
- 管理员权限

## 步骤
1. 启动 `stm32tesseract-gui`。
2. 切换到 `Environment` 栏。
3. 点击 `Up` 启动构建环境设置。
4. 此程序将自动下载必要的工具链并配置环境变量。
5. ***重启电脑以确保新的环境变量生效。***

## FAQ
### 如何排查下载失败故障？
如果下载失败，请检查网络环境并重试。

### 如何配置代理服务器？
在某些企业内部网络中，可能需要配置代理服务器来下载必要的文件。

STM32Tesseract 会自动检测并使用系统环境变量中的代理服务器设置。STM32Tesseract 将使用 `http_proxy` 和 `https_proxy` 环境变量。
