# 简介
STM32Tesseract 是一款实用程序，旨在将 STM32CubeMX 生成的代码与现代 IDE 和构建系统无缝集成。

## 组成部分
### CLI
这是 STM32Tesseract 的命令行接口，可通过终端命令直接使用，或在脚本中被调用。

## 动机
虽然 STM32CubeMX 在生成 STM32 项目方面十分出色，但它在支持用户友好的现代工具链方面往往存在不足。STM32Tesseract 旨在弥补这一差距，通过促进与高级 IDE 和构建系统的集成来增强开发体验。
