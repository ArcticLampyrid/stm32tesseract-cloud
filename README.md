# STM32Tesseract-Cloud
A serverless cloud for [STM32Tesseract](https://stm32tesseract.alampy.com/) that uses Cloudflare Workers.

## Development
```
pnpm install
pnpm run dev
```

## Deployment
```
pnpm run deploy
```
